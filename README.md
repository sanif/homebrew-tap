# Homebrew Tap

Personal Homebrew tap for tools by [@sanif](https://github.com/sanif).

## Installation

```bash
brew tap sanif/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [bytefall](https://github.com/sanif/bytefall) | Real-time network traffic visualization with Matrix-style aesthetics |

## Usage

```bash
# Install bytefall
brew install sanif/tap/bytefall

# Or after tapping
brew tap sanif/tap
brew install bytefall
```
